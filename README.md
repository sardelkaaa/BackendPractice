# Практика

Веб-сервис для приёма заявок на практику, документооборота и контроля задач практикантов по когортам.

## Стек

- **Frontend**: Next.js, TypeScript, shadcn/ui
- **Backend**: Node.js, TypeScript, Express 5, трёхслойная архитектура (controller → service → repository)
- **БД**: PostgreSQL + Prisma ORM
- **Документы**: docxtemplater
- **Контейнеризация**: Docker (frontend и backend — отдельные контейнеры)

## Роли

- **Практикант** — подаёт заявку, проходит практику, заполняет документы, отчитывается по задачам.
- **Админ** — управляет когортами, заявками, документами и задачами всех практикантов.

## Модули

1. Когорты, анкета, тестовое задание
2. Регистрация и авторизация (JWT)
3. Заявка на практику
4. Личный кабинет
5. Документы (генерация .docx)
6. Админ-панель
7. Задачи и прогресс
